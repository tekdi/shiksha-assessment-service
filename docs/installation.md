# Installation

Before setting up the Assessment service, ensure you have the following prerequisites

### Prerequisites

- [Node.js] - Version 16 or later
- [NestJS] - Familiarity with NestJS framework
- [PostgreSQL] - Database for storing test schema and attempts
- [Redis] - Cache manager for high performance

[//]: #
[Node.js]: http://nodejs.org
[NestJS]: https://docs.nestjs.com
[PostgreSQL]: https://www.postgresql.org/
[Redis]: https://redis.io/

### Setting Up Locally

Follow these steps to set up the Assessment service on your local machine:

1. **Install Node.js**  
   If you haven’t already installed Node.js, download it from the [Node.js official website](http://nodejs.org). Follow the installation instructions for your OS.

2. **Install PostgreSQL**  
   Download and install PostgreSQL from the [PostgreSQL official website](https://www.postgresql.org/download/).

3. **Install Redis**  
   Download and install Redis from the official site or run via Docker.
